# Prot

The protection flags.

## Properties

#### NONE: Prot
No protection flags

#### X: Prot
Executable flag

#### R: Prot
Read flag

#### W: Prot
Write flag

#### XR: Prot
Executable + Read flag

#### XW: Prot
Executable + Write flag

#### RW: Prot
Read + Write flag

#### XRW: Prot
Executable + Read + Write flag

## Example
```python
import mem
flag = mem.Prot.X
```
